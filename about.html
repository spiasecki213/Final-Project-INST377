About

For my website, I used the Prince George's County CRIME dataset as my source to visualize data so that current and future residents of PG County can keep track of crime in their area.

PC County CRIME Dataset

~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

Please follow the steps below to navigate the website:
Step 1: Date Filters
Click on each of the date filters and select a date you want to look at the crime data for. The filter will get dates including and in between the start and end date. The minimun date is set to April 22, 2023 due to using 1000 rows at a time, but is not the final "minimum" date due to the API updating frequently.
Step 2: Crime Type
Click on one of the radio buttons to select which crime type you would like to see the data for. By default, the crime type is set to "Select All", which will return a list of crimes of all types.
Step 3: Address
Type in the latitude and longitude of the address you want to focus around and the map will then shift its view to center around your given point. This is meant for new and existing residents to enter in their address and then check the area around them for instances of crime.
Step 4: Filter Button
Pressing the filter button will yield a list of crimes, with the ID, Date, and Crime Type fields being visible. The map will also update to house markers of each crime instance. Clicking on a marker will show you the same information as the list of crimes does. (i.e. ID, Date, and Crime Type)
